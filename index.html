Solicitar Exclusão de Conta - ShopEase

Para excluir sua conta e todos os dados associados:

  1. Envie um e-mail para suporte@appease.com com o assunto: "Exclusão de Conta".
  2. No corpo do e-mail, inclua seu e-mail cadastrado no app.

Dados excluídos:

  • Dados de autenticação (Firebase Auth).
  • Histórico de pedidos (Firebase Realtime Database).
  • Imagens de perfil (Firebase Storage).

Os dados serão removidos permanentemente em até 30 dias.
